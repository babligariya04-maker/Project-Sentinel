# 🛡️ Project Sentinel: Supply Chain Immunity

**Predicting the Ghost Bottlenecks Before They Cascade**

Modern global supply chains are highly complex but dangerously reactive. Critical transit disruptions—from sudden weather events to local political strikes—are usually identified only *after* delivery timelines are compromised. 

**Project Sentinel** acts as a supply chain immune system. It detects localized risks early and dynamically reroutes shipments before a localized bottleneck can crash the global network.

---

## 🚀 Core Innovations

* **📡 Dark Data Sensory Layer:** Scrapes unstructured, hyper-local text data (simulated alerts) to detect disruptions at "Zero-Hour" before official GPS delays occur.
* **🧠 LSTM Predictive Engine:** A deep learning time-series model that evaluates sequences of risk data to predict the exact duration of an impending delay.
* **🕸️ Contagion Graph Routing:** Treats supply chain delays as a "virus." Uses mathematical graphing (`NetworkX`) to isolate the infected node and automatically calculate dynamic bypass routes.
* **💻 Command Center UI:** An interactive Streamlit dashboard allowing logistics managers to simulate real-world disruptions and watch the AI reroute shipments in real-time.

---

## 🛠️ Tech Stack
* **Language:** Python 3.x
* **Frontend:** Streamlit, Folium (Interactive Mapping)
* **Backend Algorithms:** NetworkX (Graph Theory, Dijkstra's Algorithm)
* **Machine Learning:** TensorFlow / Keras (LSTM Architecture)
